# SignInLogs

KQL queries focused on Microsoft Entra ID authentication activity, sign-in monitoring, account abuse detection, identity risk analysis, access anomalies, and authentication security monitoring using the `SignInLogs` table.

## Available Detections

| Rule                                                                                                                                                                                                                               | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                | Blog                                                                                             |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| [Attempts to Sign In to Disabled Accounts - Entra ID](https://github.com/ITProfessorCloud/Microsoft-Sentinel/blob/main/KQL/SignInLogs/Analytic%20Rules/Attempts%20to%20Sign%20In%20to%20Disabled%20Accounts%20-%20Entra%20ID.json) | Detects sign-in attempts to disabled Microsoft Entra ID accounts (error code 50057). Events are prioritised using factors such as sign-in interactivity, time elapsed since the account was disabled, and Microsoft Entra ID sign-in risk levels. Application count is retained as a supporting risk indicator rather than a primary scoring factor, helping reduce false positives caused by benign token refresh activity following account disablement. | [Read Blog](https://www.itprofessor.cloud/fixing-attempts-to-sign-in-to-disabled-accounts-rule/) |

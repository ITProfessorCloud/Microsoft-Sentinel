# User Audit & Insider Threat Queries

Comprehensive KQL templates for user investigation and insider threat detection.

## What's Here

Production-ready queries for:
- Identity verification and profile lookups
- Sign-in activity and device logons
- Entra ID audit trails
- File access, downloads, and sharing
- Email and attachment forensics
- Insider threat patterns (data exfiltration, sabotage)

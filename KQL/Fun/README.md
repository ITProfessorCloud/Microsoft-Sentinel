# Fun Queries

Experimental KQL queries for creative data exploration and visualization.

## What's Here

**01PacManBoard.kql**: Generates ASCII art PacMan board using KQL render functions.

**02SecurityAlertsStressLevels.kql**: Categorizes alert counts with stress level indicators.

**03UserPasswordResetBehavior.kql**: Tracks password reset patterns and user behavior anomalies.
